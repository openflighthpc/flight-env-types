# Flight Environment Types

A collection of software package environment types for use with Flight
Environment.

## Overview

These types are for use with Flight Environment and provide
definitions of a number of commonly used software package
environments.

## Installation

This collection should be used with an existing functional
installation of Flight Environment. Please refer to Flight Environment
project for information on how to install.
